# Credix - Home Credit Risk LabFIA

## 1. Descrição do Projeto

O projeto Credix é uma solução de Machine Learning aplicada à análise de risco de crédito, construída a partir da base Home Credit Default Risk.

A proposta simula o ciclo real de um projeto de dados em ambiente corporativo, conectando:

- entendimento de negócio;
- engenharia de dados;
- construção de ABT;
- treinamento de modelo;
- avaliação de performance;
- deploy via API;
- interface de negócio;
- monitoramento;
- proposta de automação com agentes de IA.

---

## 2. Problema de Negócio

A Credix é uma empresa fictícia de concessão de crédito que deseja reduzir perdas financeiras causadas por inadimplência.

O desafio de negócio é identificar, antes da aprovação do crédito, quais clientes apresentam maior probabilidade de inadimplência.

---

## 3. Objetivo

Construir uma solução capaz de estimar a probabilidade de inadimplência de um cliente e apoiar a decisão de crédito por meio de uma recomendação automatizada.

A resposta do modelo é convertida em três faixas de decisão:

```text
Baixo risco
Aprovação automática sugerida

Risco médio
Enviar para análise manual
